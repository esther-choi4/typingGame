# Typing Game
The game that adapts to your typing speed. 

## Quick Start 
\# install the dependencies via npm

```
npm install

npm install node-fetch 
```
\# start the server in dev mode with HMR

```
npm run start
```
\# go to http://localhost:1234/ to run
